# High-fidelity mobile onboarding

A high-fidelity mobile onboarding UI built with React, Vite, Tailwind CSS v4, and shadcn/ui components. Originally designed in Figma.

**Live demo:** https://suhuilong.github.io/Hightsonboarding/

The original Figma project is available at https://www.figma.com/design/31r5tLMtwM1V4Lzjb0znZw/High-fidelity-mobile-onboarding--Copy---Copy-.

## Tech stack

- React 18
- Vite 6
- Tailwind CSS v4
- Radix UI / shadcn components
- MUI Icons
- Framer Motion

## Running locally

```bash
npm install
npm run dev
```

## Deployment

This project automatically deploys to GitHub Pages via GitHub Actions on every push to `main`.

To set up GitHub Pages for the first time:

1. Go to **Settings → Pages** in the GitHub repository
2. Set **Source** to **GitHub Actions**
3. Push to `main` — the workflow will build and deploy automatically

The live site will be available at `https://suhuilong.github.io/Hightsonboarding/`
